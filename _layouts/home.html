Welcome!

I also work with microservices, cloud technologies, and DevOps to create scalable AI applications. In my side projects, I've developed a large-scale search engine and a real-time face recognition system, solving challenges in data processing and computer vision.

Beyond my work, I'm passionate about contributing to the open-source community on GitHub. I'm always excited about the next public template or project release, eager to collaborate and share new innovations with the world.
